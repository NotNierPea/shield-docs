# Commands To Start Games

## Multiplayer Commands

* `Party_MinPlayers 1`
* `LobbyDedicatedSearchSkip 1`
* `LobbySearchSkip 1`
* `LobbyLaunchGame`

## Zombie Commands

* `LobbyLaunchGame`

## Blackout

* `startwzmatch` - starts match
* `set use_wz_escape 1` - change to alcatraz map day
* `set use_wz_escape_alt 1` - change to alcatraz map night
